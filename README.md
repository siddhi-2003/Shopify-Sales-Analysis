# Shopify-Sales-Analysis

## 📌 Project Overview
This project focuses on analyzing Shopify sales data to evaluate business performance and customer behavior. The dashboard provides insights into revenue trends, product performance, and regional sales distribution.

## 📊 Key Features
- KPI tracking (Revenue, Orders, AOV)
- Top-selling products analysis
- Sales trends over time
- Region-wise performance insights

## 🛠 Tools Used
- Power BI 
- Data Cleaning & Transformation
- Data Visualization

## 🚀 Outcome
Built an interactive dashboard to support data-driven decision-making and improve business performance.
